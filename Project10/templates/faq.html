❓ Frequently Asked Questions

💡 What does this app do?

This app predicts house prices using a machine learning model trained on real USA housing data.

📊 What algorithm is used?

We use Linear and Multiple Regression from scikit-learn to build predictive models.

🧠 Do I need to log in to use the predictor?

Yes, login is required to access the prediction features and personalized results.

🛡️ Is my data secure?

Yes. Passwords are securely hashed using industry standards, and sessions are protected using Flask's security features.

📈 Where is the data stored?

All data is stored in an SQLite3 database and is only used for training and prediction within this application.
