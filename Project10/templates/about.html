Welcome to Smart House Price Predictor

This project is a real-world implementation of Machine Learning and Web Development using Linear and Multiple Regression models. Trained on the USA Housing Dataset from Kaggle, it allows users to predict house prices based on demographic and structural parameters like income, house age, room count, and population.

This app is built using Flask, SQLite3, and Scikit-Learn and styled with expert-level CSS and GSAP animations.

It also includes user authentication, prediction history, data visualization, and feedback features to simulate a real-world ML SaaS app.
